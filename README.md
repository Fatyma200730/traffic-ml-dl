# Projet de détection du trafic web avec ML & DL

## Objectif
Analyser le trafic web et détecter les anomalies à l’aide du Machine Learning et du Deep Learning.

## Fonctionnalités
- Analyse du trafic web
- Détection d’anomalies (Machine Learning)
- Prédiction du trafic (Deep Learning - LSTM)

## Structure du projet
- dl-model : modèles de Deep Learning
- ml-model : modèles de Machine Learning
- backend : API et gestion des données
- data : jeux de données
- dashboard : visualisation des résultats
